Spotify Web API

An API to surface the Spotify catalog through GET requests

Search & Lookup

http://ws.spotify.com/service/version/method{.format}?parameters

http://ws.spotify.com/search/1/album.json?q=foo
http://ws.spotify.com/search/1/artist.json?q=foo
http://ws.spotify.com/search/1/track.json?q=foo

http://ws.spotify.com/lookup/1/.json?uri={uri}
  • Provides information like name, release date, popularity, artists
  • Search: String -> multiple results
  • Lookup: Spotify URI -> Details for that result

libspotify

Embed Spotify in your application or hardware.

C library wrapped in Ruby, Python, Java, C#

Builds in iOS, Android, Win32, Mac OS X/Darwin, Linux, even one for Raspberry Pi

It has enough features to write a complete client. See Viva
